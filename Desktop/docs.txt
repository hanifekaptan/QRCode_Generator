----------------------------------------------------------------------------------------------------------------------------------------

Description:

    Application that converts data from various configurations to Qr Code form.

----------------------------------------------------------------------------------------------------------------------------------------

Files in project:

    1. Main.py

    2. App.py

    3. QrCodeWindow.py (GUI)

    4. OutputWindow.py (GUI)

    5. icons_rc.py (GUI)

----------------------------------------------------------------------------------------------------------------------------------------

A. Main.py (file):

    This file contains codes that run the application.
